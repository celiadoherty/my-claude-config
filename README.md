# My Claude Config

Claude Code commands I've built and use day-to-day.

## Commands

| Command | Description |
|---|---|
| [`/gemini-meeting-notes`](commands/gemini-meeting-notes/) | Fetch Gemini meeting emails from Gmail and save as markdown |

## Installation

Copy any command's `.md` file into `~/.claude/commands/` — it's immediately available as a slash command in Claude Code.
